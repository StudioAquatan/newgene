## Dependency

- Django 2.1
- django-bootstrap4 0.0.7
- Python 3.6.5
- Arduino 1.8.7

## Setup

- 上記の環境を構築してDjangoでアプリを作成してください。
- M5Stackにm5stack_program.inoをコンパイルしてください。
